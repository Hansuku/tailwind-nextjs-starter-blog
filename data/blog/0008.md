先直接上代码：

```javascript
input[type='checkbox']{
            background-color:rgba(0,0,0,0);
            -webkit-appearance : none ;
            border:1px solid black;
            width:2rem;
            height:2rem;
            border-radius: 50%;
            outline: none;
        }
        input[type='checkbox']:checked{
            -webkit-appearance : none ;
            background: url("yourbg") no-repeat center center;
            background-size:100%;
            outline: none;
            border:none;
        }

input[type='checkbox']{
            background-color:rgba(0,0,0,0);
            -webkit-appearance : none ;
            border:1px solid black;
            width:2rem;
            height:2rem;
            border-radius: 50%;
            outline: none;
        }
        input[type='checkbox']:checked{
            -webkit-appearance : none ;
            background: url("yourbg") no-repeat center center;
            background-size:100%;
            outline: none;
            border:none;
        }
```

样式从
![](https://www.hansuku.com/wp-content/uploads/2017/12/6af346d2a68b13c103c81447ee57a051.png)
变成了
![](https://www.hansuku.com/wp-content/uploads/2017/12/adb6bc09337fc8850e55e09fc2f66845.png)
这里最主要是
`-webkit-appearance : none ;`
清除了 chrome 内核的 radio 和 checkbox 默认样式
appearance 总共有三个：

```
-webkit-appearance: value;
-moz-appearance: value;
appearance: value;
```

这样就可以更改了
